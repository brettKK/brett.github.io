ipfs 🔗

去中心化的文件系统。

怎么存（Merkle DAG, IPLD），

Blob 对象 存小文件{"data": "hello"}, List 对象 存大文件 {"data": ["blob", "list"], "links": [{"hash": "d", size: 23}, {}]}

tree 对象 代表一个目录。

commit 对象 代表根文件夹的 hash 快照， 版本控制。

怎么找（分布式哈希表 DHT）？

实现了 内容路由，节点路由，数据存储。

文件小于 k 时，DHT 存储数据。

文件大于 k 时 DHT 存 nodeid。

key 是 hash， value 是（blob，）

git 的 blob, tree, commit
