个体的行不通与行得通 🔗

自己创建作品，下面的东西行不通：

  1. 刷题。 刷到极限就是去卖课，如代码随想录。
  2. 数据库存储。 tp数据库的存储结构比较稳定，不容易创新。只能去卷ap，htap，云原生等外围生态领域。个人不能发挥决定性的作用。
  3. 云原生方面。 主要是做运维服务容器的平台，大公司在使用，小公司或者个人不使用对业务也没有影响。
  4. 重度AI训练。 数字造梦， 生成式AI、大模型需要大数据和gpu集群，成本高，对普通个人的机会几乎为0。普通人很难用chatGPT来赚钱，基本上是当韭菜被收割了，也不可能做出chatGPT, 需要上万块的GPU。

要了几天饱饭就不记得西北风啥味了
自己创建作品，下面的东西可能行得通：

  1. 视频、教程。
  2. 游戏。 godot轻量游戏引擎，开发2D游戏。godot + rust 游戏。
  3. 轻度AI训练（某个细分垂直领域的chatGPT）,web等 + rust + typescript + html + css + react。

计算机软件 = 存储（tidb，tikv，rocksdb）， 计算（zero knoledge）， 网络（tokio）
